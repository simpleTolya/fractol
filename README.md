# fractol

DRAW FRACTALS

## Launch

1. Build a project using Makefile

   ```sh
   $ make
   ```
2. Run a program with one argument (for example: mandelbrot, julia, "burning ship")

   ```sh
   $ ./mandelbrot julia
   ```
